ℹ Informazioni

Fonte dati traffico

I dati di velocità e flusso sono forniti dalle API TomTom Traffic Flow (Flow Segment Data v4). Le rilevazioni vengono effettuate su 16 posizioni lungo Viale Leonardo da Vinci (Prato), in entrambe le direzioni di marcia, fino a 14 volte al giorno negli orari di punta.

Tecnologie

TomTom Traffic API Supabase GitHub Actions Leaflet.js Chart.js GitHub Pages

Area monitorata

Viale Leonardo da Vinci (SS719), dal km 0 (Casale) al km ~5 verso Firenze, con focus sul Restringimento del Soccorso (incrocio con Via Pietro Nenni).
